# Forge2 Kanban

A Trello-style Kanban application built for the Forge2 Edition 2 Qualifier.

## Tech Stack

- Laravel 13
- React (Vite)
- SQLite
- REST API

## Features

- Create Boards
- Create Lists
- Create Cards
- Edit Cards
- Delete Cards
- Move Cards between Lists
- Assign Members
- Add Tags
- Due Dates
- Overdue Card Highlighting

## Installation

### Backend

```bash
cd backend
composer install
php artisan migrate
php artisan serve
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
